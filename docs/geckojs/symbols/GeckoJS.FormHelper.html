Class GeckoJS.FormHelper


Extends GeckoJS.Helper, GeckoJS.Singleton.
GeckoJS.FormHelper provides a set of methods that help a Controller in parsing and retrieving data from an XUL form.

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.

<static> GeckoJS.FormHelper.appendItems(el, data, mapping)
Appends items to a selection tag (select/listbox/menulist).

The items are passed in as an array. Each item consists of a label and a value. If an item is given as a string, the string is used as both label and value. Otherwise, the given mapping is applied to the item to produce a label and a value. Mapping may be a function or an object. If mapping is a function, it is invoked with the item as the single parameter, and should return an object with "value" and "label" properties. If mapping is an object, it should contain "value" and "label" properties, which in turn contain the actual value and label property names of the item.
Parameters:
{Element} el
This is a DOM element representing a selection tag
{Object} data
This is the list of items to append
{Object} mapping
This is a mapping function/object
