Class GeckoJS.Dir

GeckoJS.Dir provides a logical representation of a directory in the local file system. Through GeckoJS.Dir, local directories may be created, read, written to, queried, and otherwise manipulated.

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.

{Number} remove(aRecursive)
Removes the directory corresponding to this Dir instance.

If aRecursive is "false", then the directory must be empty before it can be deleted.

If errors occur while attempting to remove the directory, this method returns one of the following error code:
-1: path does not exist
-2: path is not a directory
-3: removal fails
Parameters:
{Boolean} aRecursive
This flag indicates if directory is to be deleted if it is not empty
Returns:
{Number} "null" if directory is successfully removed; an error code otherwise
