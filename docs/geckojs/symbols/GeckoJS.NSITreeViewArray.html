Class GeckoJS.NSITreeViewArray


Extends GeckoJS.BaseObject.
The nsITreeView interface is used by the tree widget to get information about what and how to display a tree widget. Implementing a nsITreeView in lieu of DOM methods for tree creation can improve performance dramatically, and removes the need to make changes to the tree manually when changes to the database occur.
