Class GeckoJS.Log.ClassLogger

GeckoJS.Log.ClassLogger is used to log class-specific messages. Each GeckoJS.Log.ClassLogger instance is associated with a class name and a log level. The class logger will only record log messages at or above its level, and each log entry will be tagged with its class name.
