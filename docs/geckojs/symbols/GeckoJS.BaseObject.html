Class GeckoJS.BaseObject

GeckoJS.BaseObject is the base object class upon which other GeckoJS classes are built. It defines a basic set of common methods which may be overridden by inheriting classes.

Extends GeckoJS.Class.

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.
