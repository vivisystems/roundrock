<static> {Object} GeckoJS.include_once(scriptSrc, scope)
Synchronously loads and executes the script from the specified URL in a given scope. The default scope is the GREUtils.global. If the script is executed successfully, this method returns the NS_OK result code; otherwise NS_ERROR_INVALID_ARG is returned. Once thes cript has been successfully executed, this method will not execute the script again on subsequent calls; it will simply return NS_OK. This is an alias for GREUtils.include_once()
