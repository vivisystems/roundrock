Class GeckoJS.Map

GeckoJS.Map implements the Map data structure, which is a collectin of key-value pairs containing no duplicate keys.

Extends GeckoJS.BaseObject.

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.
