Class GeckoJS.DatasourceJsonFile


Extends GeckoJS.Datasource.
GeckoJS.DatasourceJsonFile implements a data source that stores data in JSON-encoded format in a file. It also maintains a write-through data cache.

It is a Singleton class.

Data is stored in files under the "data" sub-directory of the system tmp directory (configuration key "TmpD").

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.
