Class GeckoJS.Thread

GeckoJS.Thread represents a thread of execution. It implements the GeckoJS.NSIRunnable interface, and is used to execute a GeckoJS.NSIRunnable object in a worker thread.

Extends GeckoJS.NSIRunnable.
