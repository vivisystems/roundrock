Class GeckoJS.ArrayQuery


Extends GeckoJS.BaseObject.
GeckoJS.ArrayQuery provides a set of common array handling functions useful for processing and manipulating model query results.

Data items may be accessed via their array indices.

Defined in: </home/rack/workspace/geckojs/dist/GeckoJS_docs.js>.

{Function} buildSortFunction(order)
Constructs a function that can be used to sort the elements of an array based on the given sort order.

The order is given as a string containing comma-separated element property names. The order between two elements is determined by comparing the property values in turn. Sort direction is ascending ("asc") by default. To use descending direction on a property, append the string "desc" to the property separated by a space.
