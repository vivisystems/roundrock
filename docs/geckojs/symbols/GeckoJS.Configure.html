Class GeckoJS.Configure

GeckoJS.Configure provides a Singleton configuration repository for the GeckoJS App Engine. Each configuration entry consists of a key-value pair. The configuration entries are stored in the "map" field.

The configuration repository contains a read-only "events" field of type GeckoJS.Event. This "events" field may be associated with event listeners that are invoked whenever the repository is updated. Please see the individual method description for information on the details of the various update events.

Extends GeckoJS.BaseObject, GeckoJS.Singleton.

<static> GeckoJS.Configure.clear()
Clears all entries from the configuration repository.

This method clears all key-value entries stored in the configuration repository, generating a "clear" event that is dispatched to all listeners of that event after all the entries have been cleared. This Configure instance is passed to the event listeners as the event data (in the field eventItem.data).

write(key, value, savePref)
Adds a configuration entry to the repository.

This method adds the given key-value pair to the repository. A "write" event is dispatched to all listeners of that event after the entry has been added, where the key-value pair is passed to the event listeners as named properties of the event data object (i.e. as "eventItem.data.key" and "eventItem.data.value").

Any existing entry in the repository associated with the same key is replaced with the new value.

{String} key
This is the configuration key
